1. El DOM

Nodos y jerarquia

El DOM (Document Object Model) es la forma en la que el navegador "entiende" tu página. Cuando el navegador lee tu archivo .html, no lo ve como texto plano, lo convierte internamente en una estructura de árbol de objetos que JavaScript puede leer y modificar.


El concepto de Nodo (Node)

Un Nodo es la unidad básica del árbol. Es un error pensar que solo las etiquetas HTML son nodos; absolutamente todo, es un nodo:

  • El documento completo es un nodo raíz.
  • Cada etiqueta HTML (<div>) es un nodo elemento.
  • El texto dentro de las etiquetas "Hola mundo" es un nodo de texto.
  • Los atributos (class="btn") de las etiquetas también son nodos.

La jerarquía (la estructura de árbol) es importante para entender cómo navegar y manipular el DOM.

  1. Document: La "raíz" de todo. Es el nodo superior.
  2. Parent (Padre): Es el nodo que contiene a otros.
  3. Child (Hijo): Es el nodo que está dentro de otro.
  4. Sibling (Hermano): Son nodos que tienen el mismo padre (están al mismo nivel).

    Navegar por el DOM
  1. element.parentNode (para obtener el padre)
  2. element.childNodes o element.children (para obtener todos los hijos)
  3. element.firstChild (para obtener el primer hijo)
  4. element.lastChild (para obtener el último hijo)
  5. element.nextElementSibling (para obtener el hermano siguiente)

Seleccion

Ejemplos de selección: document.getElementById(), document.querySelector(), document.querySelectorAll().

const titulo = document.getElementById('titulo-principal');

// Seleccionar por clase, devuelve HTMLCollection, colección de objetos
const elements = document.getElementsByClassName('nameClass')

// Seleccionar por etiqueta, devuelve HTMLCollection

// Seleccionar todos los elementos de un selector css (NodeList)

Iteración elementos

Es prefiere el uso de for...of sobre forEach cuando necesites usar break o trabajar con asincronía. 


// 1. Obtenemos una NodeList con todos los items
const productos = document.querySelectorAll('.producto-item');

// 2. Recorremos con for...of
for (const producto of productos) {

    // Acción: Aseguramos un estilo base
    producto.classList.add('visible');

    // Control de flujo: Si encontramos un producto agotado, 
    // lo marcamos y detenemos el bucle (ahorro de rendimiento)
    if (producto.textContent.includes('Agotado')) {
        producto.style.border = '2px solid red';
        console.log('Se encontró el primer agotado, deteniendo búsqueda...');
        break; // <--- ¡Esto es lo que forEach no puede hacer!
    }
}
