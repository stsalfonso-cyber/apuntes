Objeto JSON

JSON es un formato de texto ligero utilizado para el intercambio de datos. Aunque deriva de la sintaxis de objetos de JavaScript, es un formato de datos independiente que puede ser utilizado por casi cualquier lenguaje de programación (Python, PHP, Java, etc.).

Reglas de sintaxis básicas:

  1. Claves: Siempre deben estar entre comillas dobles ("clave").
  2. Valores: Solo se permiten cadenas (dobles comillas), números, booleanos, arrays, objetos o null. No se permiten funciones, undefined o fechas.
  3. Sin comas finales: El último par clave-valor no puede terminar con una coma.

const persona = {
    "name": "Dani",
    "age": 30,
    "isWorking": true,
    "family": ["Miguel", "Maria"],
    "address": {
        "street": "Calle de la piruleta",
        "number": 13,
        "city": "Barcelona"
    }
}

Métodos principales:

  • JSON.stringify(objeto): Convierte un objeto JavaScript en una cadena con formato JSON. Es útil para enviar datos a través de la red o para almacenar datos en formato de texto.
    • 
const persona = { name: "Dani", age: 30 };
const jsonString = JSON.stringify(persona);
console.log(jsonString); // Imprime '{"name":"Dani","age":30}'
  • JSON.parse(cadena): Convierte una cadena JSON en un objeto JavaScript. Es útil para recibir datos en formato JSON y convertirlos en objetos que se puedan manipular en JavaScript.
    • 
const jsonString = '{"name":"Dani","age":30}';
const persona = JSON.parse(jsonString);
console.log(persona.name); // Imprime "Dani"
console.log(persona.age); // Imprime 30
Consejos prácticos
  • Es conveniente envolver a JSON.parse en un bloque try...catch para manejar posibles errores de sintaxis en la cadena JSON.
  • JSON.stringify() no puede convertir funciones, undefined ni fechas.
