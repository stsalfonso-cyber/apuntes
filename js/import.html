Import/Export

Los módulos son la forma estándar de organizar el código en archivos separados. Permiten dividir una aplicación grande en piezas pequeñas, reutilizables y fáciles de gestionar, en lugar de tener miles de líneas en un único archivo.

Export: Para compartir código entre archivos, se utiliza la palabra clave export.
Puedes exportar varias cosas de un mismo archivo escribiendo export delante de cada elemento que quieras exportar:


export function miFuncion() {
    export const IVA = 0.21;
    export class Producto {
        constructor(nombre, precio) {
            this.nombre = nombre;
            this.precio = precio;
        }
}

Se utiliza default para exportar una función, variable o clase por defecto:


export default function miFuncion() {
    // código
}

Import: Para usar el código exportado en otro archivo, se utiliza la palabra clave import.

se debe especificar el nombre del elemento que se quiere importar entre llaves:


import { nombreElemento } from './miModulo.js';

Si el módulo exporta un elemento por defecto, no necesita llaves y se puede importar con cualquier nombre:

Ruta. Utiliza ./ para buscar archivos en la misma carpeta o ../ para subir un nivel. Si usas bibliotecas externas, simplemente pones el nombre (ej: import { ... } from 'framer-motion').
