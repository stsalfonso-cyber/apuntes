Estructura de datos MAP

El objeto Map es una de las estructuras de datos más potentes introducidas en ES6. Aunque a primera vista parece un objeto ({}), su comportamiento es drásticamente distinto y está diseñado específicamente para gestionar conjuntos de datos dinámicos donde necesitas asociar claves con valores.

    - Map vs Objeto:
  • Claves versátiles: A diferencia de los objetos, las claves pueden ser cualquier cosa (funciones, objetos, números, arrays, no solo strings).
  • Orden de inserción: Los elementos en un Map mantienen el orden en que fueron añadidos, lo que no ocurre con los objetos.
  • El tamaño de un Map se puede obtener con la propiedad size, mientras que en los objetos no hay una forma directa de obtener el número de propiedades.
  • Sintaxis básica: 
    
const login = new Map([   
                        ['user', 'alfonso'],
                        ['password', '123456']
                     ]); // Crea un Map con un par clave-valor inicial
    Metodos principales:
  • set(clave, valor): Agrega un nuevo par clave-valor al Map o actualiza el valor de una clave existente.
    • 
login.set("user", "alfonso"); // Agrega un nuevo par clave-valor al Map
login.set("password", "123456"); // Agrega otro par clave-valor al Map
console.log(login.get("user")); // Imprime "alfonso"
  • get(clave): Devuelve el valor asociado a la clave especificada, o undefined si la clave no existe.
    • 
console.log(login.get("user")); // Imprime "alfonso"
console.log(login.get("password")); // Imprime "123456"
console.log(login.get("email")); // Imprime undefined (clave no existe)
  • has(clave): Devuelve true si el Map contiene la clave especificada , o false si no la contiene. 
    
console.log(login.has("user")); // Imprime true
console.log(login.has("email")); // Imprime false
  • delete(clave): Elimina el par clave-valor asociado a la clave especificada. Devuelve true si la clave existía y fue eliminada, o false si la clave no existía.
