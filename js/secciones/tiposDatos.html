Tipos de datos

JavaScript tiene varios tipos de datos, incluyendo números, cadenas, booleanos, objetos y arrays.


🔸 números

  • +: suma
  • -: resta
  • *: multiplicación
  • /: división
  • %: módulo (resto de la división)
  • **: exponente

🔸 cadenas/strings

  • let nombre = "Pedro"
  • let nombre = 'Pedro'
  • let nombre = `Pedro` Esta permite ${vble}

Template strings

Permite incluir variables dentro de una cadena de texto utilizando la sintaxis ${variable}.

´el valor de la variable es: ${variable}´ (´´son tildes)


Concatenar

Para unir dos cadenas de texto, podemos usar el operador +:

let saludo = "Hola, " + nombre;

Añadir caracteres especiales. Con el carácter \ mas el símbolo

  • - let x = "Este símbolo \" es un carácter especial";
  • - El carácter \n es un salto de línea
  • - no puedes imprimir <div> de esta forma para ello tienes que escribirlo asi &lt;div>

Funciones de strings
  • - cadena.length (devuelve el nº de letras)
  • - “ingredientes”.slice(5,11) = “diente” (extrae)
  • - “patrulla”.replace('ru', 'i') = 'patilla' (remplaza)
  • - “la_casa_del_árbol”.split(' ') = [ 'la', 'casa', 'del', 'árbol' ] devuelve un array
  • - “separa”.split('') = ['s', 'e', 'p', 'a', 'r', 'a']

🔸 lógicos / booleanos

truthy = (verdadero) --- falsy = (falso)


5 > 3 // true 5 < 3 // false 5 >= 3 // true 5 >= 5 // true 5 <= 3 // false 5 <= 5 // true 5 === 5 // true 5 !== 5 // false

También es falsy

0 (cero) -0 (cero negativo)
"" (una cadena vacía) null
undefined NaN (Not a Number)

trusthy = falsy!


Operador lógico AND &&

true && true // → true true && false // → false false && false // → false Operador lógico OR || true || true // → true true || false // → true false || false // → false Operador lógico NOT ! !true // → false !false // → true
