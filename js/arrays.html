Arrays

Los arrays son estructuras de datos que almacenan una colección de elementos ordenados. En JavaScript, los arrays pueden contener cualquier tipo de dato, incluyendo otros arrays y objetos.

  • const numbers = [1, 2, 3, 4, 5];
  • const frutas = ["manzana", "banana", "cereza"];

Acceder a un elemento del array console.log(frutas[0]); // manzana

Modificar: frutas[0] = "pera";

Propiedad length: console.log(frutas.length); // 3

Método push() añade un elemento al final y devuelve la longitud

Método pop() elimina y devuelve el último elemento de un array

Método splice() Elimina o sustituye elementos en una posición específica.


const frutas = ["plátano", "fresa"]
frutas.push("naranja")
console.log(frutas) // ["plátano", "fresa", "naranja"]

Método shift() elimina y devuelve el primer elemento de un array


const frutas = ["plátano", "fresa", "naranja"]
const primeraFruta = frutas.shift()
console.log(frutas) // ["fresa", "naranja"]
console.log(primeraFruta) // "plátano"

Método unshift() añade un elemento al principio de un array.

Método concat() concatena dos array

Spread Operator ...

Permite expandir los elementos de un array dentro de otro array o dentro de una función. Es una forma más concisa y legible de concatenar arrays o pasar elementos como argumentos.

...array equivale a array[0],array[1], array[2], etc.

Transformas los elementos de un array en elementos individuales:


const numbers = [1, 2, 3]
const numbers2 = [4, 5]
//                    1, 2, 3        4, 5     
const allNumbers = [...numbers, ...numbers2]
console.log(allNumbers) // [1, 2, 3, 4, 5]

Los usos mas avanzados del spread operator son para la fusión de objetos o para el envio de argumentos a funciones.

- Para la Fusión de objetos, antes se usaba object.assign(), con '...' es mucho mas limpio:


const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const obj3 = { ...obj1, ...obj2 };
console.log(obj3) // { a: 1, b: 2, c: 3, d: 4 }

- Para el envío de argumentos a funciones, el spread operator permite pasar los elementos de un array como argumentos individuales: (evita usar apply())


const coordenadas = [10, 20]; 
ubicar(...coordenadas); // Equivale a ubicar(10, 20)

Parametro rest ...

Agrupa múltiples elementos y los convierte en un array dentro de la declaracion de una función


function sumar(...numeros) {
  return numeros.reduce((total, num) => total + num, 0);
}
console.log(sumar(1, 2, 3, 4)); // Resultado: 10

Destructuring

Puedes extraer partes de un objeto o array y "guardar lo que sobre" en una variable


const numeros = [1, 2, 3, 4, 5];
const [primero, segundo, ...resto] = numeros;
console.log(primero) // 1
console.log(segundo) // 2
console.log(resto) // [3, 4, 5]

El parametro rest ... debe ser el último parámetro de la función

A diferencia del spread operator, el parametro rest se utiliza en la definicion de la función, el spread operator se usa en la invocación de la función

En la Llamada (Invocación): ... = Spread

En la Declaración (Definición): ... = Rest


Iteración de arrays en JavaScript

Bucle while


let frutas = ['🍎', '🍌', '🍓']
let i = 0 // lo usaremos como índice
while (i < frutas.length) {
    console.log(frutas[i]) // imprime el elemento en la posición i
    i++ // incrementamos el índice en 1 para la siguiente iteración
}

Bucle for


let frutas = ['🍎', '🍌', '🍓']
for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]) // imprime el elemento en la posición i
}

for...in para objetos


const persona = { nombre: 'Alice', edad: 30, ciudad: 'Madrid' };
for (let clave in persona) {
    console.log(clave + ": " + persona[clave]) // imprime cada clave y su valor
}
// clave = nombre, edad, ciudad
// persona[clave] = Alice, 30, Madrid

forEach() método de arrays


let frutas = ['🍎', '🍌', '🍓']
frutas.forEach(function(fruta, index, originalArray){
    console.log(fruta) // imprime cada elemento del array
});
// También se puede usar con funciones flecha:
frutas.forEach(fruta => console.log(fruta));

for...of se puede usar para arrays, NodeLists (lo que obtienes con .childNodes), HTMLCollections (lo que obtienes con .children), maps, sets y strings. A diferencia del forEach, este no ejecuta una función, solo ejecuta el bloque, pero permite detener el codigo con break, continue, return e incluso admite await. ForEach no permite nada de esto


let frutas = ['🍎', '🍌', '🍓']
for (let fruta of frutas) {
    console.log(fruta) // imprime cada elemento del array
}

Arreglos bidimensionales o matrices

Un arreglo bidimensional (o matriz) en JavaScript es simplemente una estructura anidada: un arreglo donde cada elemento, a su vez, contiene otro arreglo.

🟤 Búsqueda en Arrays con sus métodos

  • .indexOf(): array.indexOf("buscar elemento") devuelve el índice de la primera aparición del elemento o -1 si no se encuentra
  • .includes(): array.includes("buscar elemento") devuelve true si el elemento existe en el array, o false si no se encuentra
  • .some(): array.some(codigo => codigo === "buscar elemento") devuelve true si al menos un elemento cumple con la condición, o false si ninguno lo hace
  • .every(): array.every(codigo => codigo === "buscar elemento") devuelve true si todos los elementos cumplen con la condición, o false si al menos uno no lo hace
  • .find(): array.find(elem => elem === "buscar elemento") devuelve el primer elemento que cumple con la condición o undefined si no se encuentra
  • .findIndex(): array.findIndex(elem => elem === "buscar elemento") devuelve el índice del primer elemento que cumple con la condición o -1 si no se encuentra
  • .sort(): array.sort() ordena los elementos del array en su lugar, modifica el array original y devuelve el array ordenado
  • .toSorted(): array.toSorted() devuelve un nuevo array con los elementos ordenados, sin modificar el array original

    • 🟤 Transformar Arrays

  • .map(): array.map(elem => elem * 2) devuelve un nuevo array con los resultados de aplicar la función a cada elemento del array original
  • .filter(): array.filter(elem => elem > 5) devuelve un nuevo array con los elementos que cumplen la condición especificada en la función
  • .reduce(): array.reduce((acumulador, elem) => acumulador + elem, 0) devuelve un único valor que resulta de aplicar la función a un acumulador y a cada elemento del array (de izquierda a derecha)
