Objetos

Los objetos en JavaScript son colecciones de propiedades, donde cada propiedad es una asociación entre un nombre (clave) y un valor. Los objetos pueden contener cualquier tipo de dato, incluyendo otros objetos y funciones.

    Contienen:
  • Propiedades: Son las características o atributos del objeto, representados como pares clave-valor. Pueden ser datos de texto, numero, etc…, arrays y otros objetos
  • Métodos: Son funciones que están asociadas a un objeto y pueden operar sobre sus propiedades.
  • Declaración: 
    
const persona = {
    name: 'Dani'   // <- variable
    age: 30,
    isWorking: true,
    family: ['Miguel', 'Maria'],
    address: { // <- Otro objeto
        street: 'Calle de la piruleta',
        number: 13,
        city: 'Barcelona'
    },
    walk: function () { // <- método
        console.log('Estoy caminando')
    }
}
  • Acceso a valores:

    console.log(persona.name); // Dani

    o

    console.log(persona["name"]); // Dani

    * también se usa corchetes si se quiere acceder a la propiedad usando una variable o si el nombre de la propiedad tiene un espacio en blanco ‘ ‘.

    Ej. const propiedad = "name"; console.log(persona[propiedad]); // Dani

  • Modificación de valores: persona.edad = 31;
  • Visualización del objeto: console.log(persona); // { nombre: "Alice", edad: 31, ciudad: "Madrid" }
  • Acceso a métodos: persona.walk() // -> Estoy caminando
  • Añadir o modificar propiedades: persona.newProperty = "value";Si el valor existe modifica su valor, y si no, lo crea

Atajos

Para crearlos con varibles que ya existen:


const name = "Dani";
const age = 30;
const persona = { name, age }; // Equivale a { name: name, age: age }

Para crear métodos sin escribir la palabra function:


const persona = {
    name: 'Dani',
    age: 30,
    walk() { // Equivale a walk: function() {...}
        console.log('Estoy caminando')
    }
}

Método destructuring: Recuperar valores de un objeto de forma sencilla


const persona = { name: 'Dani', age: 30, city: 'Barcelona' };
const { name, age } = persona; // Extrae las propiedades name y age del objeto persona
console.log(name) // Dani
console.log(age) // 30
