Estructuras de control

Permiten controlar el flujo de ejecución del código.

  • if(condición) { ... } else { ... }

    
         if (edad >= 18) {
  console.log('Eres mayor de edad')
} else if (edad >= 16) {
  console.log('Eres casi mayor de edad')
} else {
  console.log('Eres menor de edad')
}
    Las llaves {} no siempre son obligatorios. Si el bloque de código sólo tiene una línea, puedes omitir las llaves

  • El operador ternario

    Es la forma abreviada de escribir un if...else. Su estructura es: condición ? valor_si_true : valor_si_false. 

    
const esMayorDeEdad = edad >= 18 ? 'Sí' : 'No';
console.log(`¿Eres mayor de edad? ${esMayorDeEdad}`);

  • El operador && como "Cortocircuito"

    Este no es solo un operador lógico, es una técnica de renderizado condicional. Funciona así: condición && acción. Es un atajo para ejecutar un código si se cumple la condición.

    ¿Cómo funciona? Si la condición es true, ejecuta la acción. Si es false, se detiene (hace "cortocircuito") y no hace nada.

    
// Si el usuario está logueado, mostramos el botón
const estaLogueado = true;
estaLogueado && mostrarBotonCerrarSesion();

    El operador || como "Cortocircuito"

    Parecido al && pero este comprueba el primer valor y si es verdadero devuelve ese valor y si es falso devuelve el segundo

    
const carrito = JSON.parse(localStorage.getItem('carrito')) ?? [];

    Resumen:

    • Con &&: Si izquierda = falso se ejecuta derecha
    • Con ||: Si izquierda = null devuelve derecha, si no, devuelve izquierda

  • switch (expresión) { case valor1: ... break; case valor2: ... break; default: ... }

    
const dia = new Date().getDay()
switch (dia) {
   case 0:
   case 6:
      console.log("¡Hoy es fin de semana! 🥳")
      break
   case 1:
   case 2:
   case 3:
   case 4:
      console.log("¡Nooo, a trabajar! 😢")
      break
   case 5:
      console.log("¡Hoy es viernes! 🤓")
   break
}

    Es muy importante que no te olvides de poner el break al final de cada caso.

  • for (inicialización; condición; incremento) { ... }

    Ejecuta un bloque de código un número específico de veces 
    
for (let number = 1; number <= 10; number++) {
   console.log(number)
}

    Al igual que en while, for también tiene las palabras reservadas continue y break para controlar el flujo de ejecución del bucle.

  • while (condición){...}

    
let contador = 10;
while (contador>0) {
  console.log(contador);
  contador--;
}
    Ejecuta un bloque de código mientras la condición sea verdadera

    Cuando el intérprete de JavaScript encuentra la palabra break, sale del bucle y continúa ejecutando el código que haya después.

    Cuando encuentra la palabra continue, salta a la siguiente iteración del bucle.

  • do {...} WHILE (condición)

    Ejecuta un bloque de código al menos una vez y luego continúa ejecutándolo mientras la condición sea verdadera 
    let respuesta
do {
  respuesta = confirm("¿Te gusta JavaScript?");
} while (!respuesta)
