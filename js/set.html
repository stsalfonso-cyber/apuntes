Estructura de datos SET

Conjunto de valores únicos. Su característica principal es que no se puede repetir ningún elemento


const conjunto = new Set([1 ,8, 2, 7 ,8, 1, 2, 3])
console.log(conjunto)   // 1, 8, 2, 7, 3

conjunto.add(elemnt) añade si no está

conjunto.delete(elemnt) elimina si está

conjunto.has(elemnt) devuelve true si el elemento está en el conjunto, o false si no lo está

conjunto.size devuelve el número de elementos únicos en el conjunto


🟤Acceso

Lo mas común es acceder por iteración con los bucles for of y forEach


const miSet = new Set(['Spagnolo', 'Moda', 'España']);
for (const valor of miSet) {
     console.log(valor);
}

Otra manera es la conversión a Array y acceder por índice. Para convertir a Array se usa el spread operator


const arr = [...miSet];
console.log(arr[0]); // 'Spagnolo'

Los principales usos de un Set son para eliminar duplicados de un array o para verificar la existencia de un elemento en una colección.

Importante. No se puede acceder a los elementos de un Set por índice, ya que no tienen un orden definido.
