Variables y Tipos de datos

Tipos Escalares

  • Strings: (Cadenas) "Hola".
  • Integers: (Números enteros) $edad = 25.
  • Floats: (Números decimales) $precio = 19.99.
  • Booleans: (Verdadero/Falso) $activo = true.
  • Null: Representa una variable sin valor o inexistente.

La importancia de los "Falsy" (Valores falsos)

Variables ($variable) Se pueden declarar e inicializar al vuelo con el símbolo $. Son dinámicas (aunque apliquemos tipado estricto en funciones).

Constantes (const IVA) Se definen fuera o dentro de clases y no llevan $ delante.


En PHP, estos valores son considerados false (falsy):

  1. false (el booleano en sí).
  2. 0 (integer) y 0.0 (float)
  3. "0" (el string que contiene el número cero). ¡Cuidado! Este es el mayor peligro en PHP.
  4. "" (string vacío).
  5. null.
  6. [] (array vacío). En JavaScript, [] (array vacío) es true.

El operador ??

$variable = (variable o expresión) ?? (Valor si lo de derecha no existe o es null)

Es equivalente al operador || de JS pero mas seguro porque solo actua cuando el valor es realmente null o undefined evitando falsos positivos con otros valores falsy

    
// Si $_GET['nombre'] existe, úsalo; si es null, usa 'Invitado'.
$nombre = $_GET['nombre'] ?? 'Invitado';

Arrays Asociativos

En un array normal (indexado), el acceso es por número: $colores[0] = "rojo".

En un Array Asociativo, el acceso es por una clave (string): $datos['nombre'] = "Alfonso". En JS tambien existía el método objeto.campo en PHP no se puede hacer de este modo


Sintaxis

Se usa el operador de asignación flecha =>.

    
$usuario = [
    'nombre' => 'Alfonso',
    'rol'    => 'admin',
    'edad'   => 25
];

// Acceso y modificación
echo $usuario['nombre']; // Imprime: Alfonso
$usuario['rol'] = 'editor'; // Modificar

Por qué son el "Objeto" de PHP

Aunque PHP tiene Clases y Objetos, durante años (y en muchos frameworks), los desarrolladores han usado arrays asociativos para todo porque son extremadamente ligeros. Si recibes una fila de una base de datos (ej: una fila de la tabla productos del inventario de Spagnolo), lo más normal es que PHP te la devuelva como un array asociativo.


La iteración

En JS usas .map() o .forEach(). En PHP, la construcción reina es foreach

    
  foreach ($usuario as $clave => $valor) {
    echo "La propiedad $clave tiene el valor: $valor";
}

▫ $clave: Es el nombre del índice ('nombre', 'rol', 'edad').

▫ $valor: Es el contenido ('Alfonso', 'admin', 25)..


Funciones últiles

  1. count($array): Devuelve el número de elementos.
  2. array_keys($array): Devuelve solo las claves (['nombre', 'rol', 'edad']).
  3. array_values($array): Devuelve solo los valores (['Alfonso', 'admin', 25]).
  4. in_array($valor, $array): Comprueba si un valor existe dentro del array.

Estructuras de control

  1. if / elseif / else 
            
if ($edad >= 18) {
    echo "Mayor de edad";
} elseif ($edad >= 16) {
    echo "Casi mayor";
} else {
    echo "Menor de edad";
}
  2. switch (¡Cuidado con la trampa!)

    Al igual que en JS, el switch compara valores. Pero en PHP el break es obligatorio si no quieres que el código siga ejecutándose hacia abajo.

  3. La estrella moderna: match (PHP 8+)

    El match es la evolución del switch introducida en PHP 8. Es mucho más seguro y limpio.

    ▪ Diferencia clave: A diferencia del switch, match devuelve un valor (es una expresión), no necesita break y es estricto con los tipos de datos.

            
$resultado = match ($rol) {
    'admin'  => "Acceso total",
    'editor' => "Acceso limitado",
    default  => "Invitado",
};

echo $resultado; // Devuelve el valor directamente

El operador ternario

$variable = (condición) ? valor_si_verdadero : valor_si_falsy;

    
$edad = 18;
$mensaje = ($edad >= 18) ? "Eres mayor de edad" : "Eres menor de edad";

echo $mensaje; // Imprime: Eres mayor de edad

Arrays iteraciones

foreach

A diferencia de JS en PHP el foreach no es un método del array sino una funcion que recibe el array como argumento. Olvídate del for, en PHP, el 99% de las iteraciones se manejan mejor con el foreach, ya que gestiona igual de bien arrays normales como asociativos

  • Sintaxis normal: 
            
$frutas = ["🍎", "🍌", "🍓"]; 
foreach ($frutas as $fruta) { 
  echo $fruta; 
}
  • Sintaxis clave-valor: 
            
$usuario = ['nombre' => 'Alfonso', 'edad' => 25]; 
foreach ($usuario as $key => $value) { 
  echo "$key es $value"; 
}

array_map, array_filter, array_reduce

PHP tiene equivalentes a los métodos de JS, pero con un "pero" técnico: el nombre de las funciones empieza por array_, reciben una funcion de primer argumento y el array como segundo argumento


array_map (Transformación)

    
// En JS: arr.map(x => x * 2)
$numeros = [1, 2, 3];
$dobles = array_map(fn($n) => $n * 2, $numeros);

array_filter (Filtrado)

    
// En JS: arr.filter(x => x > 2)
$numeros = [1, 2, 3, 4];
$mayores = array_filter($numeros, fn($n) => $n > 2);

array_reduce (Acumulación)

    
// En JS: arr.reduce((acc, x) => acc + x, 0)
$numeros = [1, 2, 3];
$total = array_reduce($numeros, fn($acc, $n) => $acc + $n, 0);
