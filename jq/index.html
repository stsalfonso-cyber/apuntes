Apuntes de jQuery

Guía completa basada en el documento original

1. Introducción y carga

Se requiere incluir la librería jQuery antes de empezar. Es fundamental esperar a que el DOM esté completamente cargado para evitar errores de ejecución.

<script src="https://code.jquery.com/jquery-3.4.1.min.js"></script>

2. Selectores comunes

jQuery permite seleccionar elementos usando sintaxis CSS y selectores propios.

$("#id") -> Seleccionar por ID.

$("elemento") -> Seleccionar por tipo (ej: $("p")).

$(".clase") -> Seleccionar por clase CSS.

$(".container li:first") -> Pseudoselectores (hijos).

3. Manipulación de contenido

text(valor): Lee o establece el texto de uno o más elementos.

let x = $("#parrafo1"); x.text("Este es el nuevo texto");

html(valor): Establece contenido HTML, permitiendo inyectar tags. 

$("#div").html('<input type="text">');

4. CSS y Atributos

Para estilos: .css(propiedad, valor) o recibir un objeto para múltiples.

Nombre de propiedades css, Acepta guiones o camelCase:
$(elemento).css('background-color','white')
$(elemento).css('backgroundColor','white') 

$(el).css({color: 'red', width: '50px'});

Para atributos: attr(prop, valor) y removeAttr(prop).

5. Administración de eventos

jQuery simplifica el manejo de eventos (click, mouseover, hover, focus, blur, dblclick, etc.).

  • ready: Función que se ejecuta cuando el DOM está completamente cargado.
    $(document).ready(function() {...})
  • click: Función que se ejecuta cuando se hace click en el elemento.
    $(documento).click(nombre de función)
  • hover: Función que se ejecuta cuando se pasa el ratón por encima del elemento.
    $(documento).hover(nombre de función)
  • mousemove(e): Útil para coordenadas e.clientX.
    $(documento).mousemove(nombre de función)
  • focus / blur: Gestión del foco en campos de formulario (focus cuando tiene el foco, blur cuando pierde el foco).
    $(documento).focus(nombre de función)
  • this : Referencia al elemento que disparó el evento.
    $(this).css("color", "red")

6. Manipulación del DOM

Métodos para insertar y eliminar elementos:

  • empty(): Borra el contenido interno. $(documento).empty()
  • append() / prepend(): Añadir al final o al principio.
    $(documento).append("contenido")
    $(documento).prepend("contenido")
  • eq(index).remove(): Eliminar nodo específico por índice. $(documento).eq(index).remove()
  • slice(0, 2).remove(): Eliminar un rango de elementos. $(documento).slice(0, 2).remove()

7. Efectos Visuales

Métodos para animaciones rápidas:

show() / hide()

$(documento).show() Oculta un elemento lentamente

$(documento).hide() Muestra un elemento lentamente

fadeIn() / fadeOut()

$(documento).fadeIn() Oculta y muestra un elemento lentamente con opacidad

$(documento).fadeOut() Oculta el elemento

fadeTo()

$(documento).fadeTo(speed, opacity) Oculta y muestra un elemento lentamente. El método fadeTo puede modificar la opacidad de un elemento, el efecto es llevar la opacidad actual hasta el valor que le pasamos al método fadeTo

$(documento).fadeTo() Oculta el elemento.

toggle()

$(documento).toggle() Oculta y muestra un elemento lentamente

$(documento).toggle() Oculta el elemento.

slideDown() / slideUp()

$(documento).slideDown(speed, callback) Oculta y muestra un elemento lentamente

$(documento).slideUp(speed, callback) Oculta el elemento.

8. Iteración por los elementos (each)

jQuery dispone de un método que nos permite asociar una función que se ejecutará por cada elemento que contenga la lista del objeto jQuery

    <ul id="mi-lista">
      <li>Elemento 1</li>
      <li>Elemento 2</li>
      <li>Elemento 3</li>
    </ul>
    <script>
      $('#mi-lista li').each(function (index) {
        console.log("Índice: " + index + ", Texto: " + $(this).text());
      }
    );
    </script>

9. AJAX

jQuery maneja peticiones asíncronas de forma muy sencilla:

  • load(url, params, callback)

    Este método permite cargar datos desde el servidor y actualizar el contenido de un elemento HTML sin necesidad de recargar toda la página

    $("#contenedor").load("archivo.html");
  • ajaxStart / ajaxStop

    Son eventos globales en jQuery que permiten gestionar indicadores de carga (spinners) de forma automática

    $(document).ajaxStart(function() {
       $('#spinner-cargando').show();
    });
  • $.get() / $.post(): Peticiones HTTP básicas. // Obtener información de un usuario
    $.get("https://example.com", function(respuesta) {
       // Código que se ejecuta si la petición tiene éxito
       console.log("Nombre: " + respuesta.nombre);
       $("#resultado").html("Usuario cargado: " + respuesta.nombre);
    }).fail(function() {
       console.log("Error al cargar los datos.");
    });
  • $.getJSON()

    Se le puede pasar un objeto JSON o una url que devuelva un JSON. Convierte automáticamente el objeto JSON en un objeto JavaScript, facilitando su manipulación.

    $.getJSON('https://api.tuweb.com/datos', function(data) {
       // Procesamos el objeto 'data' recibido
       let htmlContenido = `

          Perfil: ${data.nombre}

          Edad: ${data.edad}

          Rol: ${data.rol}

          Habilidades: ${data.habilidades.join(', ')}

       `;
       $('#resultado').html(htmlContenido);
    }).fail(function() {
       alert("Error al cargar el archivo JSON.");
    });
  • $.ajax(config): La configuración completa (async, type, beforeSend, success, etc).

10. Metodos addClass y removeClass

Métodos para añadir y quitar clases CSS:

addClass() $(elemento).addClass("recuadro")

removeClass() $(elemento).removeClass("recuadro")

toggleClass() $(elemento).toggleClass("recuadro")
